# React-TicTacToe

Basic Tic Tac Toe game built using React with Next.js and typescript.

## Main Concepts

In the process of creating the project, the following concepts have been applied or utilized:

### React

1. Components
2. JSX
3. Conditional rendering
4. Event Handling
5. States
6. Hooks
